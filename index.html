How it works

Bring your own workbook.

01

Get the official SCF

Download the free Excel workbook from the Secure Controls Framework Council — the licensing stays theirs, the data stays yours.

02

Drop it in

A Web Worker parses every sheet in seconds and caches the model locally in IndexedDB. Nothing is uploaded — there's nowhere to upload it to.

03

Explore everything

Controls, crosswalks, maturity levels, risks, threats, assessment objectives, evidence — indexed, searchable, linked.

Private by architecture

Your workbook never leaves your browser.

No backend

Static files on GitHub Pages. There is no server to send your data to.

No accounts

No sign-up, no login, no tracking pixels, no telemetry of any kind.

Local cache

Parsed once into IndexedDB — instant reloads, workbook stays on disk.

Works offline

Installable PWA. After first load, the whole app runs without a network.

Open source

MIT-licensed, built in the open.

The entire viewer — parser, indexes, UI — is free software. Read the code, file issues, send PRs. If it helps your compliance program, star it and tell a colleague.
